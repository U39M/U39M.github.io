トップページ作成

中華料理屋「大垣飯店」

解説

職業訓練の課題として制作したものです。課題では、架空の中華料理店「大垣飯店」を題材に設定し、そのトップページを作成いたしました。

制作にあたっては、既存のPhotoshopによるデザインカンプを参考とし、デザインを忠実に再現する形でWebページへと実装を行いました。これにより、ビジュアルデザインからコーディングへとつなげる実務的な流れを学習することを意図しております。
目的

地域に根ざした中華料理店として「大垣飯店」の魅力を広く伝えることにあります。料理の特徴や店内の雰囲気を発信し、新規のお客様に来店を促すとともに、常連のお客様にも安心感と親しみを持っていただける情報発信の場とすることを意図しております。

また、トップページを通じて店舗の存在感を高め、地域に愛されるお店としてのブランドイメージを強化することも目的のひとつとなっております。
ターゲット

「大垣飯店」を利用する地域の一般顧客を想定しております。特に、家族連れや学生、会社員など、幅広い世代の方々に親しんでいただける中華料理店をイメージしております。
対応サイズ PC / スマートフォン
使用ツール HTML / CSS / JavaScript / Adobe Photoshop
