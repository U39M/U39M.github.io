HP/お問い合わせフォーム作成

Web制作広告「HOT TOWN WEB」

解説

職業訓練の課題として制作したものです。岐阜高島屋の閉店後に岐阜駅周辺や柳ケ瀬商店街を盛り上げることを目的としたチラシの内容を参考にし、それをWebサイトとして再構築いたしました。

また、Webデザインの訓練生が地域のお店の活性化に貢献するというテーマのもとで作成しており、シンプルな構成の中にお問い合わせフォームも含め、1ページ完結型のデザインを採用しています。
目的

職業訓練の一環としてWeb制作の実践力を身につけるとともに、実際の地域活性化活動を模した学習を行うことにあります。あわせて、訓練生が「企業やお店のWebサイトを無料で制作する」というテーマのもとで活動を展開するイメージを提示し、地域社会への貢献の可能性を示すことを意図しております。
ターゲット

地域の商店や中小企業の方々を主な対象としております。特に、デジタル媒体を活用した情報発信に関心を持ちながらも、自社サイトの制作や運営に課題を抱えている方々を想定しております。
対応サイズ PC
使用ツール HTML / CSS / JavaScript(AOS.js) / PHP / Adobe Illustrator
